    1.1.5 DataCleanManager(本应用数据清除管理器)

     1）cleanApplicationData(Context context, String... file path)

    清除本应用所有的数据 

 2）cleanCustomCache(String filePath)

    清除自定义路径下的文件，使用需小心，请不要误删。而且只支持目录下的文件删除

  3）cleanDatabases(Context context)

    清除本应用所有数据库(/data/data/com.xxx.xxx/databases)

  4）cleanDatabaseByName(Context context, String dbName)

    按名字清除本应用数据库 * * 

  5）cleanExternalCache(Context context)

    清除外部cache下的内容(/mnt/sdcard/android/data/com.xxx.xxx/cache)

  6）cleanFiles(Context context)

    清除/data/data/com.xxx.xxx/files下的内容 

  7）deleteFilesByDirectory(File directory)

    删除方法 这里只会删除某个文件夹下的文件，如果传入的directory是个文件，将不做处理

  8）cleanInternalCache(Context context)

     清除本应用内部缓存(/data/data/com.xxx.xxx/cache)

  9）cleanSharedPreference(Context context)

    清除本应用SharedPreference
